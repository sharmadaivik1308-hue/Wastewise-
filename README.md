# WasteWise (AI Smart Waste Segregation System)

## 1. Title and Description
WasteWise is an AI-powered waste management system that allows users to scan or upload images of waste items and get instant classification along with proper disposal instructions.

## 2. Features
- AI-based waste detection (YOLO model)
- Real-time camera scanning
- Waste classification (organic, recyclable, hazardous)
- Disposal suggestions
- Reward points system
- User-friendly interface

## 3. Tech Stack
**Frontend:**
- HTML
- CSS
- JavaScript

**Backend:**
- Python (Flask)

**AI/ML:**
- TensorFlow / YOLO model

**Deployment:**
- GitHub Pages (Frontend)
- Local/Cloud backend

## 4. Project Structure
- `frontend/`: Contains the user interface built with HTML, CSS, and JavaScript.
- `backend/`: Includes the Flask server and API endpoints.
- `model/`: Stores the YOLO/TensorFlow models for waste classification.
- `assets/`: Holds static resources and images used in the project.

## 5. How to Run the Project

**Backend:**
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run Flask server:
   ```bash
   python app.py
   ```

**Frontend:**
- Open `index.html` in your web browser OR run using a live server extension in your code editor.
